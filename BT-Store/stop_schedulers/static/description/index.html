Stop all the schedulers

And enable them again without having to remember which ones were active.

Without this app

Without this app, if you needed to stop all the schedulers because of a maintenance, or because something went wrong and you wanted to set the system in a passive way while having a look, you would need to go the the list of schedulers, write down which ones were active, and then disable them one by one. And more importantly, you would need to remember to set them active later.

With this app

With this app, you can easily go to the general configuration of your system and choose between three options to inhibit all the schedulers of the system, depending on your needs.

You can either:

Do not stop the schedulers: By selecting No, the schedulers behave normally, that is: if a scheduler is set as active, it will execute as expected, and if it is inactive, it will not execute.

Stop all the schedulers: By selecting Yes, indefinitely, the schedulers will not execute, no matter if they are set as active. To make them execute again as normal, simply set the option as No the next time.

Stop all the schedulers for a moment: By selecting Yes, until the moment indicated, the schedulers will not execute until the moment indicated in the field Date & Time is reached, and from that moment on, they will execute again as normal.
